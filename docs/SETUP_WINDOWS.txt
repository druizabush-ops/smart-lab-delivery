Название: Настройка и проверка проекта в Windows 10 (PowerShell)
Дата актуализации: 2026-04-13

1. Требования
- Windows 10
- PowerShell 5.1+ или PowerShell 7+
- Git

2. Клонирование проекта
```powershell
git clone <URL_РЕПОЗИТОРИЯ>
Set-Location .\smart-lab-delivery
```

3. Проверка структуры инженерного каркаса
```powershell
Test-Path .\docs
Test-Path .\src
Test-Path .\tests
Test-Path .\scripts
Test-Path .\config
Get-ChildItem .\docs -Name
Get-ChildItem .\src -Recurse -Directory | Select-Object FullName
Get-ChildItem .\tests -Recurse -Directory | Select-Object FullName
Get-ChildItem .\scripts -Name
Get-ChildItem .\config -Name
```

4. Проверка согласованности обязательных документов
```powershell
Test-Path .\README.md
Test-Path .\docs\ARCHITECTURE_OVERVIEW.txt
Test-Path .\docs\PROJECT_JOURNAL.txt
Test-Path .\docs\STATUS_RULES.txt
Test-Path .\docs\DECISIONS.txt
Test-Path .\docs\TECH_DEBT.txt
Get-Content .\README.md
Get-Content .\docs\ARCHITECTURE_OVERVIEW.txt
Get-Content .\docs\PROJECT_JOURNAL.txt
Get-Content .\docs\STATUS_RULES.txt
Get-Content .\docs\DECISIONS.txt
Get-Content .\docs\TECH_DEBT.txt
```

5. Проверка статуса Git
```powershell
git status
git log --oneline -n 5
```

6. Рекомендация по кодировке для русских файлов
```powershell
# Рекомендуется сохранять .txt/.md в UTF-8
[Console]::OutputEncoding = [System.Text.Encoding]::UTF8
```

7. Текущий запуск приложения
На текущем этапе полноценный runtime отсутствует: проект находится в фазе инженерной и архитектурной инициализации.

8. Что проверять на этом этапе
- Наличие обязательных документов в docs/.
- Соответствие структуры каталогов архитектурным слоям.
- Наличие записи в PROJECT_JOURNAL о произведенных изменениях.
- Отсутствие преждевременной реализации интеграционных клиентов, БД, UI и доставки.
